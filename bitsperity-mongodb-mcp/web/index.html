üìñ What is MCP?

Model Context Protocol (MCP) enables AI assistants to securely connect to external data sources and tools.


This MongoDB MCP Server allows AI models to:

üîó Connect
Connect to MongoDB databases
üîç Query
Query collections and analyze data

üõ†Ô∏è Available Tools

  • establish_connection
    Connect to a MongoDB instance using connection string
  • list_databases
    List all databases in the MongoDB connection
  • list_collections
    List collections in a specific database
  • query_collection
    Query documents from a collection with filters
  • get_collection_schema
    Analyze and return collection schema structure
  • aggregate_collection
    Run complex aggregation pipelines
  • get_sample_documents
    Get sample documents from collections
  • test_connection
    Test if database connection is active
